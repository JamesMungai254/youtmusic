
# 🎵 YouTube to Audio Converter (Streamlit)

A modern, aesthetic **Streamlit web application** that converts YouTube videos into high-quality audio files (MP3).  
The app features a **dark UI**, **video thumbnail previews**, **real-time progress tracking**, and an **instant download option**.

---

## ✨ Features

- 🔗 Convert YouTube links to MP3 audio
- 🖼 Video thumbnail & metadata preview
- 📊 Real-time download & conversion progress bar
- 🎧 High-quality audio extraction (192 kbps)
- 🌙 Dark-mode, aesthetic UI
- ⬇️ One-click audio download
- 🧩 Scalable architecture (ready for backend expansion)

---

## 🛠 Tech Stack

- **Frontend:** Streamlit
- **Media Processing:** yt-dlp
- **Audio Conversion:** FFmpeg
- **Language:** Python 3.9+

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/JamesMungai254/youtmusic.git
cd youtmusic
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

Or manually:

```bash
pip install streamlit yt-dlp requests
```

---

### 3️⃣ Install FFmpeg (Required)

#### Windows

* Download from [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
* Add FFmpeg to **System PATH**

#### Linux

```bash
sudo apt install ffmpeg
```

#### macOS

```bash
brew install ffmpeg
```

---

### 4️⃣ Run the application

```bash
streamlit run app.py
```

The app will open automatically in your browser.
