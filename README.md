# 🎵 YouTube to Audio Converter (Streamlit)

A modern, aesthetic **Streamlit web application** that converts YouTube videos into high-quality audio files (MP3).  
The app features a **dark UI**, **video thumbnail previews**, **real-time progress tracking**, and an **instant download option**.

---

## ✨ Features

- 🔗 Convert YouTube links to MP3 audio
- 🖼 Video thumbnail & metadata preview
- 📊 Real-time download & conversion progress bar
- 🎧 High-quality audio extraction (192 kbps)
- 🌙 Dark-mode, aesthetic UI
- ⬇️ One-click audio download
- 🧩 Scalable architecture (ready for backend expansion)

---

## 🛠 Tech Stack

- **Frontend:** Streamlit
- **Media Processing:** yt-dlp
- **Audio Conversion:** FFmpeg
- **Language:** Python 3.9+
